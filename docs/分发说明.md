## ⚠️ 注意事项

1. **首次运行可能被杀毒软件拦截**
   - 这是正常的，因为这是未签名的exe文件
   - 用户需要添加信任/白名单

2. **Windows系统要求**
   - Windows 7/8/10/11
   - 64位系统

3. **PDF文件要求**
   - 需要有可提取的文本层
   - 扫描件无法识别

4. **依赖安装**
   - 程序首次运行会自动安装依赖（pdfplumber）
   - 需要网络连接

## 📁 开发者版本（Python源代码）

如果用户想自己运行源代码，需要：
1. 安装 Python 3.9+
2. 安装依赖：`pip install -r requirements.txt`
3. 运行：`python gui.py`
