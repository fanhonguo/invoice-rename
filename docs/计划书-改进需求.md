## 改进需求

### 1. 去除文件名中的空格
**当前:** `美设 1657614422.pdf`
**目标:** `美设1657614422.pdf`

### 2. 修复销售方名称抓取问题
**问题:** 有时销售方名称抓取不到

### 3. 添加购买方简称作为第一项
**新格式:** `{购买方简称}{销售方名称}{发票号码}.pdf`

**简称规则:**
- `上海昊奕佳国际物流有限公司` → `昊奕佳`
- `上海璟易科国际物流有限公司` → `璟易科`

**示例:** `昊奕佳美设1657614422.pdf`

### 4. 添加 USD 后缀
**条件:** 备注中包含以下任一关键词时，在文件名末尾添加 `-USD`

**关键词列表:**
- 汇率
- $
- 仅限
- 只接受
- 美金付款 / 美元付款
- 美金支付 / 美元支付
- 支付美金 / 支付美元

**示例:** `昊奕佳美速3943794031-USD.pdf`
